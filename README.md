# California House Price Prediction API

A machine learning REST API that predicts California house prices using a Random Forest Regressor, built with **scikit-learn** and served with **FastAPI**. Supports both single predictions and bulk predictions via CSV upload.

## 📌 Overview

This project trains a regression model on the classic **California Housing dataset** and exposes it through a FastAPI web service. Users can:
- Get a single house price prediction via a JSON request
- Upload a CSV of multiple houses and get back a CSV of predictions
- Check API health and model metadata

## 🧠 Model Details

- **Algorithm:** Random Forest Regressor (`n_estimators=100`)
- **Dataset:** [California Housing dataset](https://scikit-learn.org/stable/datasets/real_world.html#california-housing-dataset) (from `sklearn.datasets`)
- **Train/Test Split:** 80/20 (`random_state=42`)
- **Average Error (MAE):** ~$39,000 - $49,000 depending on run
- **Target:** Median house value for California districts (in units of $100,000)

### Features Used

| Feature | Description |
|---|---|
| `MedInc` | Median income of the neighbourhood |
| `HouseAge` | Average age of houses in the block |
| `AveRooms` | Average number of rooms per house |
| `AveBedrms` | Average number of bedrooms per house |
| `Population` | Total population of the block |
| `AveOccup` | Average number of people per house |
| `Latitude` | Latitude of the block |
| `Longitude` | Longitude of the block |

## 🚀 Usage

### 1. Train the model

```bash
python train.py
```

This downloads the California Housing dataset, trains a `RandomForestRegressor`, prints the Mean Absolute Error and R² score, and saves two files:
- `house_model.joblib` — the trained model
- `house_features.joblib` — the list of feature names

### 2. Explore the data (optional)

```bash
python explore.py
```

Prints the shape, first few rows, and summary statistics of the dataset.

### 3. Run the API

```bash
uvicorn main:app --reload
```

The API will be available at `http://127.0.0.1:8000`. Interactive docs are auto-generated at `http://127.0.0.1:8000/docs`.

## 📡 API Endpoints

### `GET /`
Returns basic API info.

```json
{
  "message": "California house prediction api",
  "status": "running",
  "endpoint": "send POST request to /predict"
}
```

### `GET /health`
Returns model health and metadata.

```json
{
  "status": "running",
  "model": "RandomForestRegressor",
  "features": ["MedInc", "HouseAge", "AveRooms", "AveBedrms", "Population", "AveOccup", "Latitude", "Longitude"],
  "avg_error": "$39,000"
}
```

### `POST /predict`
Predicts the price of a single house.

**Request body:**
```json
{
  "MedInc": 8.3,
  "HouseAge": 41,
  "AveRooms": 6.98,
  "AveBedrms": 1.02,
  "Population": 322,
  "AveOccup": 2.55,
  "Latitude": 37.88,
  "Longitude": -122.23
}
```

**Response:**
```json
{
  "predicted_price": "$452,300",
  "predicted_price_short": "$4.52 hundred thousands",
  "confidence_range": "$413,300 to $491,300"
}
```

> **Note:** Latitude must be between 32 and 42, and Longitude between -125 and -114, since these are the geographic bounds of California covered by the training data.

### `POST /predict-file`
Upload a CSV file containing multiple houses and receive a CSV with an added `predicted_columns_usd` column.

**Required CSV columns:**
```
MedInc, HouseAge, AveRooms, AveBedrms, Population, AveOccup, Latitude, Longitude
```

**Example request (using `curl`):**
```bash
curl -X POST "http://127.0.0.1:8000/predict-file" \
  -F "file=@houses.csv" \
  --output predictions.csv
```
